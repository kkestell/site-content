Not BASIC, but close enough.

```
# This is a comment.
A = 1
B = 3.14
C = TRUE
D = FALSE
E = "Moon"
PRINT("Goodnight {E}")
IF A == 1 THEN
  FOR I = 1 TO 10 THEN
    PRINT(I)
  END FOR
END IF
F = 10
WHILE F > 0 THEN
  PRINT(F)
  F = F - 1
END WHILE
```
